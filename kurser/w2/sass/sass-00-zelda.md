## scss/partials/_mixins.scss

```scss []
@mixin heartpixelcolors ($heart-map) {
  $heart-colors: (
    t: transparent,
    g: $heart-black,
    r: $heart-red,
    w: $heart-white
  );

  $num: 1;

  @each $value in $heart-map {
    .px:nth-child(#{$num}) {
      background-color: map-get($map: $heart-colors, $key: $value );
    }
    $num: $num + 1;
  }

  .px {
    width: $px-size;
    height: $px-size;
  }
}
```
