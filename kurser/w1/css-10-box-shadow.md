# Box-shadow

Med egenskapen `box-shadow` kan vi lägga till skuggor.


```css [ ]
.selektor {
  box-shadow: inset offset-x offset-y blur-radius spread-radius color
}
```

---

```html
<div class="ex"></div>
```

```css [ ]
div {
  background-color: powderblue;
  height: 100px;
  width: 400px;
}
```

---

## Ex 1

```css [ ]
.ex {
  /* offset-x offset-y color */
  box-shadow: 20px 40px black;
}
```
